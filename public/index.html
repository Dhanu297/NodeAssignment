Personal Finance REST API

This API allows you to manage users, find his incomes and expenses using Firestore Database.

Available Endpoints

Users

  • GET /users - Get all users
  • GET /users/:id - Get user by ID
  • POST /users - Create a new user
  • PUT /users/:id - Update user details
  • DELETE /users/:id - Delete a user

Incomes

  • GET /incomes - Get all income records
  • GET /incomes/:id - Get income by ID
  • POST /incomes - Add new income entry
  • PUT /incomes/:id - Update income entry
  • DELETE /incomes/:id - Delete income entry

Expenses

  • GET /expenses - Get all expenses
  • GET /expenses/:id - Get expense by ID
  • POST /expenses - Add new expense entry
  • PUT /expenses/:id - Update expense entry
  • DELETE /expenses/:id - Delete expense entry
